About layouts

I've grouped and sorted example layouts which can be achieved with the template, but you can easily create custom ones too. All of the ready ones are also prepared for customization, so you can easily change width of the site, sidebars widths, sliders sizes, number of columns etc. But don't be overwhelmed, the defaults are very nice, and if you want something more, but you're in a hurry - I've prepared a nice set of various layouts.
There are 2 main concepts to choose from:

  • Boxed layout

  • Open layout

Each of them can be used to compose any layout:
